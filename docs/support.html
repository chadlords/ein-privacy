Getting Started

1. Create your Provider profile with your Local ID.

2. Add children, parents/guardians, and service coordinators.

3. Start a new session and use Apple Pencil to write directly on the form.

4. Save and export PDFs or monthly ZIP reports as needed. Please make certain you have configured your mail client to send emails from your device.

Frequently Asked Questions

Where is my data stored?

All data is stored locally on your device and in your private iCloud account. The developer has no access to your data.

Does the app work offline?

Yes. You can create and edit sessions completely offline. Changes will sync via iCloud when you're back online.

How do I export reports?

From any session or child record, tap the email button to generate a PDF and open the mail interface. For monthly reporting, navigate to the Provider screen (click the gear, then click Provider) use the Bulk Export feature to create a password-protected ZIP file. The Bulk Export interface will provide a To/From date range selector to generate PDFs for the archive.

Can I use this on multiple devices?

Yes. This app uses iCloud sync. Your data will appear on all devices signed into the same Apple ID.

How do I set up my Local ID Prefix?

Upon first use, you enter your Local ID Prefix. Additionally, you can edit the prefix in the Provider tab of the app configuration screen. It will be concatenated with the 5-digit number assigned to each case.
